# AZ-104 Labs

This section documents hands-on labs completed while studying for the Microsoft Azure Administrator (AZ-104) certification.

The focus of these labs is building practical experience with Azure administration, identity management, networking, governance, storage, and compute resources.

## Topics Covered

* Microsoft Entra ID
* Azure Resource Groups
* Role-Based Access Control (RBAC)
* Azure Policy
* Virtual Networks (VNets)
* Network Security Groups (NSGs)
* Azure DNS
* Azure Bastion
* Azure Storage Accounts
* Azure Virtual Machines
* Monitoring and Governance

## Purpose

To translate certification knowledge into practical cloud administration experience through hands-on implementation and documentation.
